## citation_formatting_rules

Specifies the HTML markup that must be used to represent citations: a `<mark>` with `data-type="citation"` (or equivalent) wrapping the cited text plus one or more `<sup>` elements carrying `data-citation-id`, `data-id`, `data-type`, `data-title`, and `data-reasoning` metadata. Includes rules for lists, hyperlinks, zero-nesting, and flattening of inner marks.
