## citations_section_layout_rules

Defines the optional trailing Citations section structure for agents that use a footer: its HTML skeleton (`<hr>`, heading, and list of numbered category lines), ordering requirements, and the rule that only actually-used categories appear. For the current brief agent, this is mostly historical as inline-only provenance is preferred.
